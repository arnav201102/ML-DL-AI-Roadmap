### 1. What is a linear system?

### 2. Matrix form of a system?

### 3. What is an augmented matrix?

### 4. Name elementary row operations.

Reduced Row Echelon Form.

### 7. Why do row operations preserve solutions?

They represent equivalent equations.

When an inconsistent row appears.

Free variables.
